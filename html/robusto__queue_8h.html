void robusto_set_queue_state_result ( queue_state *  state,
rob_ret_val_t  return_value 
)

Sets a Robusto flag to failed .

The Robusto queue state exploits the fact that eading and writing a single byte is an atomic operation on all relevant architectures So the queue is get a very light-weight semaphore that can bring with it a 16-bit integer rob_ret_val_t.

Parameters
flagA pointer to the flag
return_valueA pointer to where to store the return value. Can be null.
