uint16_t calc_deka_ms_since ( uint64_t  since,
uint64_t  curr_time 
)

Calculate, in tenths of milliseconds, the time since the provided time.

Note
It is in tenths as that is the default precision of the repeater and provides the most data with the least bytes.
Parameters
sinceSince when to calculate.
Returns
uint16_t The time in tenths of milliseconds, will be UINT16_MAX if more than that.
